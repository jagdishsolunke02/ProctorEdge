# 🚀 ProctorEdge – AI-Powered Online Exam Proctoring System

## 📌 About the Project

**ProctorEdge** is a smart AI-based online examination proctoring system designed to detect and prevent cheating during remote exams.
With cheating cases increasing rapidly, traditional human monitoring is no longer enough. ProctorEdge provides a secure, automated, and intelligent solution for fair online assessments.

---

## 🎯 Key Features

### 👤 Face Verification & Identity Check

* Verifies student identity using webcam
* Prevents impersonation during exams
* Detects if a second person enters the frame

### 👀 Eye Movement Tracking

* Monitors eye movement in real time
* Detects suspicious looking away patterns
* Helps identify answer searching behavior

### 📱 Unauthorized Object Detection

* Detects hidden mobile phones

### 📱 Unauthorized Object Detection

* Detects hidden mobile phones
* Identifies books, notes, and cheating materials
* Flags earphones and other restricted items

### 🎤 Audio Monitoring

* Monitors room sound through microphone
* Detects talking, whispering, and unusual noise
* Ensures a quiet and fair exam environment

### 🔒 Secure Browser Lockdown

* Prevents tab switching
* Disables copy-paste functionality
* Blocks right-click actions
* Restricts minimizing the exam window

### 📸 Live Alerts & Evidence Capture

* Sends instant alerts for suspicious activity
* Automatically captures screenshots as proof
* Stores evidence for review by authorities

### 📊 Risk Score System

* Generates cheating probability score
* Helps teachers identify high-risk candidates
* Makes final review faster and easier
